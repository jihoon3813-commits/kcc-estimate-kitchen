# KCC글라스 HomeCC 고객용 모바일 견적서

## 프로젝트 개요
KCC글라스 HomeCC 주방가구 견적서를 **카카오톡/문자 공유용 프리미엄 모바일 제안서**로 변환한 정적 웹페이지입니다.  
기존 표 기반 견적서를 고객 관점의 감성적이고 신뢰감 있는 제안서 형태로 재구성했습니다.

---

## ✅ 현재 구현된 기능

### 디자인 & UX
- **모바일 퍼스트** 반응형 레이아웃 (최적화 기준: 375~480px)
- **프리미엄 제안서 스타일**: 네이비 + 골드 투톤 컬러 시스템
- **히어로 섹션**: 브랜드명 + 총 견적가 강조 표시
- **카드형 UI**: 표 대신 읽기 편한 정보 카드 구성
- **섹션 분리**: 요약 → 시공조건 → 컨셉 → 제품 → 마감재 → 금액 → 담당자 → CTA

### 인터랙션 & 애니메이션
- **스크롤 페이드인**: 섹션 진입 시 부드러운 등장 애니메이션
- **숫자 카운트업**: 히어로 영역 진입 시 총 금액 숫자 카운트 애니메이션
- **플로팅 CTA 버튼**: 상담 전화 바로 연결, CTA 섹션 진입 시 자동 숨김
- **제품 가로 스크롤**: 기기 카드 좌우 스와이프
- **헤더 스크롤 효과**: 스크롤 시 그림자 효과

### 영업/CTA
- **상담 전화 연결** (1877-8204)
- **문자 문의** (010-8098-6606)
- **견적서 공유** (Web Share API + 클립보드 폴백)
- **담당자 카드** (최현지/김민성 전화 연결)

### 포함 정보
- 견적번호, 작성일, 유효기간, 시공예정일
- 현장 주소, 파트너명, 영업소, 담당자
- 시공 현장 조건 (엘리베이터/사다리차/철거)
- 디자인 컨셉 (미스티그레이/진그레이/크리스탈 12T)
- 4종 기기 카드 (인덕션/후드/수전/싱크볼)
- 금액 상세 (자재비/시공비/타일/물류/VAT/총액)
- 견적 안내 사항

## 🔧 추천 다음 개발 단계

1. **실제 제품 이미지 적용**: KCC HomeCC 공식 제품 이미지로 교체
2. **고객명 동적 삽입**: URL 파라미터로 고객명 개인화 (`?name=홍길동`)
3. **견적서 PDF 저장**: html2canvas + jsPDF 라이브러리 활용
4. **카카오 공유 API**: 카카오톡 공유 미리보기 최적화
5. **다국어 지원**: 영문 버전 추가 고려
6. **견적 승인/거절 버튼**: 고객 응답 수집 기능
